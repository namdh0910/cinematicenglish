# Retention Design Skill | Behavioral Psychology

## Core Philosophy
Retaining a user isn't about points; it's about **identity transformation**. We move the user from "I am trying to learn English" to "I am a cinematic learner."

## Habit Loops (The Hook Model)
1. **Trigger**: 
   - External: Cinematic notifications (e.g., "A new chapter of your story awaits.")
   - Internal: Boredom or the desire for immersion.
2. **Action**: Start a lesson (low friction, high reward).
3. **Variable Reward**: 
   - New story plot points (Cliffhangers).
   - Random "Glow" XP drops.
   - Character feedback.
4. **Investment**: 
   - Streak consistency.
   - Vocabulary vault.
   - Skill mastery levels.

## Premium Gamification Rules
- **No Confetti**: Use "Glow Transitions" and "Particle Flows."
- **No Badges**: Use "Mastery Seals" or "Relics."
- **Sound Design**: Low-frequency, deep tones for completion, not "ding-ding" sounds.
- **Visuals**: Use dark glassmorphism and gold/violet accents to signal prestige.

## Binge Learning Momentum
- **The "Next Story" Auto-Queue**: 5-second countdown to the next part.
- **Cliffhanger Hooks**: End every story with an unanswered emotional question or a language challenge.
- **Seamless Transitions**: No "Loading..." screens. Use animated background transitions that keep the user in the "flow state."
