# Personalization Intelligence Layer | Blueprint

This system defines the "Emotional Brain" of Cinematic English, ensuring the platform evolves alongside the user's identity and emotional state.

## 1. Emotional Profile Architecture (Task 1)
Every user is represented by an evolving `EmotionalProfile` that tracks:
- **Dominant Mood**: The core resonance (Void, Pulse, Calm).
- **Trait Matrix**: 
    - *Reflective*: Depth of journal entries and contemplation.
    - *Ambitious*: Engagement with power-themed content.
    - *Vulnerable*: Willingness to share emotional breakthroughs.
    - *Confident*: Measured through speaking stability and speed.
- **Pacing Preference**: Dynamic adjustment of UI transitions and story auto-queuing.

## 2. Adaptive Systems (Tasks 2, 3, 4)
| System | Input Signal | Adaptive Response |
| :--- | :--- | :--- |
| **Story Weaver** | Recent Tags + Mood | Shifts story selection towards current emotional resonance. |
| **Atmosphere Layer** | Profile traits | Adjusts blur strength, grain opacity, and glow intensity. |
| **AI Coach** | Confidence level | Switches between 'Supportive' (Heart) and 'Challenging' (Zap) tones. |
| **Momentum Engine** | Binge behavior | Speeds up transitions for "Flow" users; adds breathing pauses for "Reflective" users. |

## 3. Memory-Aware Intelligence (Task 6)
The system uses the **Cinematic Journal** history to:
- **Surface Insights**: "You hesitated less today than you did in Part I."
- **Nostalgic Reconnection**: Suggesting a story that mirrors an emotional state the user conquered months ago.
- **Identity Reinforcement**: Changing UI micro-copy (e.g., from "Ready to learn?" to "Ready to direct your next story?").

## 4. Privacy & Ethics (Task 9)
- **Surveillance-Free Tone**: Personalization is presented as "Supportive Discovery" rather than "Targeted Analysis."
- **Data Dignity**: Users have full control over their journal and reflections. No raw behavioral data is exposed to the user, only the positive evolution of their identity.

## 5. Future Evolution (Task 10)
- **Voice Bio-marker Analysis**: Using pitch and resonance to detect real-time emotional state.
- **Personalized AI Narrators**: Matching the story's narrator voice to the user's specific learning goal (e.g., authoritative vs. empathetic).
