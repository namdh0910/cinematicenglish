# Smart Recommendation Engine | Narrative Logic

## Goal
To sequence stories in a way that feels like a curated cinematic experience tailored to the user's emotional state and learning level.

## 1. Emotional Sequencing (The Flow State)
The engine prioritizes stories based on "Mood Matching":
- **Pattern**: If user completes a *Tense/High Energy* business story, the next recommendation should be *Analytical/Steady* to avoid fatigue, or another *High Energy* one if it's part of a series.
- **Category Continuity**: Stay within the same theme (e.g., "The Mafia Trilogy") to maintain narrative immersion.

## 2. Progressive Intensity
- **Complexity**: Gradually increase the vocabulary density (CEFR level) throughout a binge session. 
- **Attention Span**: Start with a 15-minute deep-dive, follow with 5-minute "Micro-stories" to maintain momentum when fatigue kicks in.

## 3. Micro Cliffhanger System
Every story end-point must contain a "Curiosity Trigger" injected into the next story's preview:
- **The "Missing Key"**: *"You've learned the words. Now, learn the silence. The next story starts now."*
- **The "Unanswered Question"**: *"But what did the CEO say next? Find out in Part 2."*

## 4. Binge Pattern Detection
If the user completes 3 stories in a row, the engine unlocks a "Mastery Challenge" — a special 2-minute high-intensity speaking task with double XP.
