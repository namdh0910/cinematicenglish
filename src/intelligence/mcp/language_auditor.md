# Language Auditor | Cinematic English MCP

This auditor ensures that every word in the platform adheres to the Cinematic Identity and avoids "cringe" or "generic" educational patterns.

## 1. The "Cringe" Detection
- [ ] **No Over-Excitement**: Are there excessive exclamation marks? (Reject > 1 per screen).
- [ ] **No Generic Praise**: Avoid "Awesome!", "Good job!", "Way to go!".
- [ ] **No Corporate Jargon**: Avoid "User experience," "Optimization," "Deliverables."
- [ ] **No Childishness**: Avoid icons or words that feel like a game for children.

## 2. Cinematic Restraint
- [ ] **Word Count**: Can this line be shorter and more powerful?
- [ ] **Subtext**: Does the copy imply an emotion rather than stating it bluntly?
- [ ] **Pacing**: Is the microcopy balanced with the atmospheric visuals?

## 3. Identity Alignment
- [ ] **Archetype Consistency**: Does the AI Coach sound like a Director/Mentor?
- [ ] **Transformation Focus**: Does the copy emphasize "becoming" rather than just "learning"?
- [ ] **Premium Feel**: Do the buttons feel substantial? (e.g., "Preserve Memory" vs "Save").

---
**REJECTION CRITERIA**:
- Any text that sounds like it could be in a standard language app.
- Aggressive motivational spam.
- Inconsistent tone between the story mood and the UI prompts.
- Using "Lesson," "Quiz," or "Exercise" in the main user journey.
